Reflection Space

By Kevin Lim, Myron Liu

Abstract

We implemented Reflection Space Image-Base Rendering as described by Cabral et al. in Reflection space image based rendering. HDR environment maps contain high-quality lighting information, but integrating them is costly, especially if view-dependent materials are involved. Our goal is to generate a scene in which the user can rotate around the object lit by an HDR environment map, and observe view dependent BRDF changes in real-time.

Description

We've tested our implementation on an Intel i5 CPU, 8GB Memory, NVIDIA GeForce 560 Ti Graphics Card machine. Our Diffuse and Phong pre-integration took 2557.1 seconds and 3521.7 seconds for a 3072 pixels by 1536 pixels HDR Grace Cathedral map, provided by the USC HDR gallery.

The Cook-Torrance integration was performed on a reduced map size of 100 pixels by 50 pixels, as our roughness value would be a high 0.1, and the shading will become blurred. This took an average 2.63 seconds per map, and for the twelve viewpoints, this took 31.6 seconds total.
